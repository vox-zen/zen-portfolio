# Zen Portfolio

Personal portfolio website built with modern web technologies.

## 🌐 Live Demo
https://zen-portfolio.fly.dev

## ⚡ Tech Stack
- Next.js
- TypeScript
- Tailwind CSS
- Docker

## ✨ Features
- Clean and minimal UI
- Responsive design
- Fast performance
- Easy to customize

## 📦 Installation

```bash
git clone https://github.com/vox-zen/zen-portfolio.git
cd zen-portfolio
npm install
npm run dev

```

## 🐳 Run with Docker
```bash
docker build -t zen-portfolio .
docker run -p 3000:3000 zen-portfolio
```
